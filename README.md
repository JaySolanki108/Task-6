# Task 6 – Password Strength Analysis

This repository contains the work completed for **Task 6** of the Cyber Security Internship, which focuses on understanding password security and evaluating password strength using an online password strength checking tool.

## Objective
The objective of this task is to analyze how password length and complexity affect security by testing multiple passwords with varying strength levels and documenting the results.

## Tool Used
- PasswordMonster (Online Password Strength Checker)

## Task Activities
- Created multiple passwords with different complexity levels
- Tested each password using an online password strength checker
- Recorded password strength levels and estimated time to crack
- Analyzed the impact of password complexity on security
- Documented findings in a formal report

## Repository Contents
- **Task-6_Password-Strength-Analysis_Report.pdf** – Detailed report of the password evaluation
- **screenshots/** – Evidence screenshots of password strength testing
- **README.md** – Overview and description of the task

## Key Concepts Covered
- Password strength
- Brute force attack
- Dictionary attack
- Password complexity
- Authentication best practices

## Outcome
This task improved understanding of secure password creation, common password attacks, and best practices for protecting accounts from unauthorized access.
